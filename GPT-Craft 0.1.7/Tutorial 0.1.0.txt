Step-1
Open CMD
Step-2
Place the folder in the Downloads directory
Step-3 
Type "cd Downloads"
Step-4
Now, type "cd GPT-Craft 0.1.7"

Tutorial on Game Controls

W=Move Forward
S=Move Backward
A=Move Left
D=Move Right
Spacebar=Jump
LMB=Break Blocks
RMB=Place Blocks
